Quick Start Examples

1. Get API Specification

curl -X GET "http://localhost:5000/api-docs" \
  -H "accept: application/json"

2. Get Current Theme

curl -X GET "http://localhost:5000/api/theme/info" \
  -H "accept: application/json"

3. Run Tests

curl -X POST "http://localhost:5000/api/test/run" \
  -H "Content-Type: application/json" \
  -d '{
    "test_type": "all",
    "parallel": true
  }'

4. Get Questions

curl -X GET "http://localhost:5000/api/questions?page=1&limit=10" \
  -H "accept: application/json"
